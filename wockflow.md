# Aho-Corasick Algorithm Visualizer

## Описание алгоритма
Алгоритм Ахо-Корасик — эффективный алгоритм поиска всех вхождений множества строк-образцов в заданную строку.

## Структура проекта

### 📁 src/
- **core/** - ядро алгоритма
  - **automaton/** - классы автомата
  - **builder/** - построители
- **visualization/** - визуализация
  - **renderers/** - рендереры
  - **themes/** - темы оформления
- **ui/** - пользовательский интерфейс
  - **commands/** - команды
- **logger/** - логирование
- **utils/** - утилиты

### 🔧 Команды
1. 📝 Add pattern - добавить образец
2. 🔗 Build - построить fail-ссылки
3. 🔍 Search - поиск в тексте
4. 🌳 Visualize - визуализировать автомат
5. 📊 Stats - статистика
6. 🔄 Reset - сброс
7. 📋 Logs - просмотр логов
8. ❌ Exit - выход

### 🎨 Визуализация
- ◆ - корень
- ● - узел
- ★ - терминальный узел
- ↯ - fail-ссылка
